## Descripción General
Este proyecto es un **diseño de tarjeta de 3 columnas responsive** que se adapta a diferentes tamaños de pantalla.  
Cada tarjeta contiene un ícono, un título, una descripción y un botón de llamada a la acción con efectos hover.

## El Desafío
Los usuarios deberían poder:

- Ver el diseño óptimo según el tamaño de pantalla de su dispositivo.  
- Observar los estados hover en elementos interactivos.

## Mi Proceso
- Usé Flexbox para alinear las tarjetas en filas.  
- Implementé estados hover para botones y tarjetas.  
- Organicé el CSS utilizando variables para colores y espaciados.  
- Hice que el diseño fuera responsive para diferentes tamaños de pantalla.

## Tecnologías Usadas
- **HTML5**  
- **CSS3**  
- **Flexbox**  
- **Propiedades personalizadas de CSS**

## Lo Que Aprendí
- Cómo estructurar un **diseño de tarjetas responsive** usando Flexbox.  
- Cómo implementar **efectos hover consistentes** usando transiciones CSS.  
- Cómo usar **propiedades personalizadas de CSS** para simplificar la gestión de colores y espaciados.
