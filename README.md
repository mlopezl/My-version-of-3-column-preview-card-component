## Overview
This project is a responsive 3-column card layout that adapts to different screen sizes. Each card contains an icon, a title, a description, and a call-to-action button with hover effects.

## The challenge
Users should be able to:
- View the optimal layout depending on their device's screen size.
- See hover states for interactive elements.

## My process
- Used Flexbox to align the cards in rows.
- Implemented hover states for buttons and cards.
- Organized CSS using variables for colors and spacing.
- Made the layout responsive for different screen sizes.

## Built with
- HTML5
- CSS3
- Flexbox
- CSS custom properties

## What I learned
- How to structure a responsive card layout with Flexbox.
- Implementing consistent hover effects using CSS transitions.
- Using CSS custom properties to simplify color and spacing management.
